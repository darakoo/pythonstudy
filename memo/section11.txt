### 목차 ###
01. 사용자 함수
    - 사용자 함수란 사용자가 직접 만드는  함수

02. 사용자 함수 용어
    1. 함수 용어 정리
    - 인수, 매개변수, 반환값

    2. 함수 정의
    def 함수이름(매개변수):
        기능구현
        return 반환값

    3. 함수 호출
    함수이름(인수)
    변수 = 함수이름(인수)

03. 인수(argument)와 매개변수(parameter)
    1. 매개변수
    2. 가변 매개변수
    3. 디폴트 매개변수(기본 매개변수)

04. 반환값
    - 반환값이란 함수 호출 결과값 또는 return 값을 말한다.

    1. 반환값이 있는 함수
    2. 다중 반환
    3. 함수의 종료를 위한 return

05. 지역변수와 전역변수
    1. 지역변수(local variable)
    - 함수 내부에서 선언한 변수이며 함수 내부에서만 사용할 수 있다.
    2. 전역변수(global variable)
    - 함수 외부에서 선언한 변수이며 함수 내부, 외부에서 모두 사용할 수 있다.


### MISSION ###
- def 키워드를 통해서 새로운 함수를 정의할 수 있다.
- 함수에 전달하는 값을 인수라고 한다.
- 인수를 저장하는 변수를 매개변수라고 한다.
- 함수의 변환값이 존재하는 경우 return문을 통해 반환할 수 있다.
- 반환값이 없는 함수에서 return문을 사용하면 함수를 종료할 수 있다.

### 상세 ###
1. 사용자 함수 목적
	1. 소스코드 재사용(함수의 기능에 특정성을 부여해서는 안된다)
	2. 소스코드 간결화

2. 사용자 함수(두 정수의 덧셈) 만들기 순서
    1) 함수 이름을 생각한다.
        def add():

    2) 매개변수를 생각한다.
        def add(num1, num2):

    3) 실행할 문장을 생각한다.
        def add(num1, num2):
            result = num1 + num2

    4) 리턴 값을 생각한다.
        def add(num1, num2):
            result = num1 + num2
            return result
